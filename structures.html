Main Structures

Standard (only content)

The .MainBox class is used in a div (usually followed by other classes) that wraps around every content that isn't the page's Toolbar, Header and Footer elements.

Most pages have the .MainBox class.

.MainBox
HTML Code Example

<!DOCTYPE html>

<!-- Simple structure example showing how the <div class="MainBox"> element is nested-->
<body class="Visible-ScreenXL">
    <div id="AppWrapper">
        <div id="Header"></div>
        <div id="HeaderToolBar"></div>

        <div class="MainBox">

        </div>

        <div id="Footer"></div>
    </div>
</body>

Action bar + Content

This structure can be most commonly found in the Admin Area.

.MainBox .LayoutFixedSidebar .SidebarFirst .ActionsSideBar
.SidebarColumn .ActionsSideBarComp
.ContentColumn
HTML Code Example
<!DOCTYPE html>

<div class="MainBox ARIARoleMain LayoutFixedSidebar SidebarFirst ActionsSideBar">
    <div class="SidebarColumn ActionsSideBarComp"></div>
    <div class="ContentColumn"></div>
</div>
